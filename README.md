# Список персонажей из Наруто

## Функционал

- Вывод персонажей из Наруто из API
- Фильтрация персонажей по инпуту
- Пагинация списка персонажей
- Роутинг страниц

## Технологии

- React
- Typescript
- React Router
- styled-components
- AntDesign (взял иконки)
- Pre-commit hook (Husky + lint-staged)
